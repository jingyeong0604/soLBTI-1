재미있는 테스트로 소비성향 알아보고 카드 추천부터 가입까지!
- 성향별/연령대별/성별 소비패턴 분석
- 맞춤형 카드 추천
- 카드 조회 및 가입

![카드 가입](https://github.com/shinhan-academy-teams/soLBTI/assets/80742079/f0346c72-da41-4fe3-aefb-17e137151941)
